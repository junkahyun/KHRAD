# KH-RAD
KH RAD는 국비지원 학원인 KH정보교육원 임직원들이 사용하는 업무용 웹사이트입니다. 
KH정보교육원에서 백엔드 웹개발자로 근무하면서 운영/배포/개발하였던 페이지를 폴더별로 정리하여 업로드해 놓았습니다. 

<br>
